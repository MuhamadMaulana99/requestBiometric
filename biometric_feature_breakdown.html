# Feature Input Output Notes
1
Extraction
 image template Converts raw biometric image into a compact, comparable feature template
2a
Identification
Identify one (1:N)
 image or template person data Returns best match from the enrolled database
2b
Identification
Identify many (1:N top-k)
 image / template k: number persons data[] Returns ranked list of top-k candidate matches with similarity scores
3a
Verification
Verify by NIK (1:1)
 NIK image / template match: bool score Confirms claimed identity against a specific enrolled record by national ID
3b
Verification
Verify biometric (1:1)
 image vs image or image vs tmpl or tmpl vs tmpl match: bool score Direct comparison of two biometric samples without involving the database
4
Quality check
 image quality score params Assesses image quality (sharpness, contrast, lighting, resolution) before processing
5
Registration
 NIK / person ID image / template enrollment ID status Enrolls a new biometric record; should run quality check and deduplication first
6
Update biometric
 NIK / person ID new image / template status Replaces or appends a new biometric sample to an existing enrolled record
7
Delete biometric
 NIK / person ID status Removes enrolled biometric data; important for GDPR / right-to-erasure compliance
8
Deduplication check
 image / template duplicate flag matched IDs[] Checks if a biometric already exists in the database before enrollment to prevent duplicates
